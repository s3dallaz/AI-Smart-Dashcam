# AI Smart Dashcam

A Computer Vision project that performs real-time object detection on driving videos using YOLOv8 and OpenCV.

## Features

* Vehicle Detection
* Pedestrian Detection
* Traffic Light Detection
* Bounding Box Visualization
* Confidence Scores
* Video Processing

## Technologies Used

* Python
* OpenCV
* Ultralytics YOLOv8
* PyTorch

## Project Structure

AI-Smart-Dashcam/

├── assets/

├── videos/

## Installation

Install dependencies:

pip install -r requirements.txt

## Usage

Run:

python main.py
